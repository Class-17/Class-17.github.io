链式线性表的基本操作及其在多项式运算上的应用

课题

设计实现两个链式储存的一元多项式加法与乘法算法，并分析算法的时间复杂度。

相应OJ: 一元多项式的乘法与加法运算

课题解析

问题分析：
由于多项式可能非常稀疏，所以宜采用链式线性表示，仅储存非零项。

加法运算：首先建立加法结果线性表，然后直接对两多项式处理。

处理过程主要分两种情况：

  1. 两多项式当前项指数不相等，将较大的一项作为结果加在结果多项式后。
  2. 两多项式当前指数相等，系数相加，将结果加在结果多项式后。若系数相加结果为零，则跳过此节点。

乘法运算：建立乘法结果线性表，采用直接运算，逐项插入方式处理两多项式。

将多项式 P2 的每一项分别与 P1 多项式相乘，将所乘形成的新项插入到中间结果多项式中。该中间结果多项式一开始为空，并以指数递减的顺序维持当前的运算中间状态。当有新项需要插入时，相当于在一个递减链表中插入一个新节点，并维持递减顺序。
A. 如果插入的新节点的指数与链表中某节点的指数一样，则将他们的系数相加；
B. 如果系数相加后的结果为零，则从中间结果链表中删除相应节点;
C. 如果不存在指数相同的节点，则将新节点插入到相应位置。

多项式读入

多项式读入方法：

  1. 首先使用整形变量 n 读入多项式项数，依据项数信息建立链式表。
  2. 建立单向链表头节点，使用 Attach 函数依次把读入的项接在链表后。
  3. 读入完毕，链表建立完成，释放头节点内存，返回多项式链表头。

多项式加法与乘法运算主体算法

多项式加法算法实现

实现要点：

  1. 当两多项式都未读取完成时，依次比较两多项式当前项：
    A. 两项不相等时，将较大的一项接在加法结果多项式后。
    B. 两项相等时，将两项系数相加，同当前项系数一同接在结果多项式后。
  2. 当有一个多项式读取完成时，将另一没有完成读取的多项式之后的项使用 Attach 函数接到结果多项式之后。
  3. 释放加法结果多项式头节点所占内存，返回结果多项式链表头。

多项式乘法算法实现

实现要点：

  1. 首先以 P1 多项式的首项与 P2 多项式的第一项相乘，初始化乘法结果多项式。
  2. 依次将 P1 多项式的各项与 P2 多项式的各项相乘结果插入到结果多项式中：
    A. 获取 P1 多项式当前项，获取 P2 多项式当前项。
    B. 根据两多项式当前项乘积结果指数，遍历乘法结果多项式，找到插入位置。
    C. 若当前项系数与待处理项指数相等，两者系数相加，改变结果多项式当前项系数。若相加后结果为 0，则删除该节点。
    D. 若当前项系数小于待处理项数指数，则将待处理项系数作为新节点插入结果多项式。
  3. 释放结果乘法结果多项式头节点，返回结果多项式链表头。
