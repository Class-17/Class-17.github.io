链式线性表的基本操作及其在多项式运算上的应用

课题

设计实现两个练是储存的一元多项式加法与乘法算法，并分析算法的时间复杂度。

相应OJ: 一元多项式的乘法与加法运算

课题解析

问题分析：
由于多项式可能非常稀疏，所以宜采用链式线性表示，仅储存非零项。

加法运算：首先建立加法结果线性表，然后直接对两多项式处理。

处理过程主要分两种情况：

  1. 两多项式当前项指数不相等，将较大的一项作为结果加在结果多项式后。
  2. 两多项式当前指数相等，系数相加，将结果夹在结果多项式后。若系数相加结果为零，则跳过此节点。

乘法运算：建立乘法结果线性表，采用直接运算，逐项插入方式处理两多项式。

将多项式P2的每一项分别与P1多项式相乘，将所乘形成的新项插入到中间结果多项式中。该中间结果多项式一开始为空，并以指数递减的顺序维持当前的运算中间状态。当有新项需要插入时，相当于在一个递减链表钟插入一个新节点，并维持递减顺序。
如果插入的新节点的指数与链表中某节点的指数一样，则将他们的系数相加；
如果系数相加后的结果为零，则从中间结果链表钟删除相应节点。
如果不存在指数相同的节点，则将新节点插入到相应位置。
